Listas ordenadas

Na lista ordenada, como o nome já diz, a ordem dos itens é importante.
Para criar uma lista ordenada utilizamos a tag <ol></ol> e para as linhas da listas utilizamos a tag <li> (Não é necessário fechar a tag).

Na lista ordenada ele numera os itens, e caso eu precise inserir um item novo no meio da lista, ele atualiza todos os numeros de todas as linhas.

Exemplo:

Quando eu acordo eu faço:
  1. Tomar café
  2. Escovar os dentes
  3. Tomar banho
  4. Me trocar
  5. Ir pra Fatec
É possível alterar os índices por outros caracteres utilizando a tag <ol type="opção desejada">. Por padrão ele escolhe a opção "1" ou numeração. (Exemplo acima).

Listas não ordenadas

Lista onde, como o nome diz, a ordem não é importante.
Para criar uma lista não ordenada, utiizamos a tag <ul> </ul>

Exemplo:

Lista de compras:
  • Pão
  • Manteiga
  • Leite
  • Arroz
  • Feijão

É possível alterar o tipo de marcador, mas por padrão ele vem com a bolinha (exemplo acima).
