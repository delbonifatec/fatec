Tags para formatação de texto

Negrito ou strong

Para colocar em negrito utilizamos a tag não semântica <b> </b> ou a tag semântica <strong> </strong>

Itálico ou emphasis

Para colocar em itálico utilizamos a tag não semântica <i> </i > ou a tag semântica <em> </em > (emphasis)

Texto Marca texto

Para colocar marca texto utilizamos a tag <mark> </mark >
Para mudar a cor, temos que utilizar CSS <mark style="background-color: cor desejada"> </mark >. Caso utilize assim no corpo do texto, se criar um style, ele não vai mudar

Criação de estilo para que possa utilizar várias vezes

Ao invés de colocar as tags acima no texto, podemos criar um estilo no head para que possamos reutilizar.
<style mark {background-color: cor desejada}> </style>. Ele vai alterar todos os marcadores que não tenha sido inserida cor.

Texto excluído (taxado) e texto inserido (sublinhado)

Para taxar um texto, utilizamos a tag <del> </del>
Para sublinhar um texto, utilizamos a tag <ins> </ins>. Há também a tag não semântica <u> </u>

Subscrito e sobrescrito

Para sobrescrever um texto utilizamos a tag <sub> </sub>. Exemplo 22=4.
Para subscrever um texto, utilizamos a tag <sub> </sub>. Exemplo H2O.

Fonte monoespaçada (mesmo espaço utiizado para cada fonte)

Exemplo: No comando document.getElementById('teste') algumas letras ocupam mais espaço que as demais, por exemplo o l ocupa menos espaço que o g, tornando assim difícil a leitura de um código

Para resolver este problema, utilizamos a tag <code> </code>. Exemplo: document.getElementById('teste')

Este comando é curto e está em apenas 1 linha. Quando formos fazer o mesmo com códigos ocupando mais de uma linha, utiizamos a tag <pre> </pre>, assim ele considera todos os espaços e quebra de linha utilizada no código na página do navegador.

Exemplo sem <pre> </pre>

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número é par {num} é PAR') else: print(f'O número {num} é IMPAR')

Exemplo com <pre> </pre>

                         
                num = int(input('Digite um número'))
                if num % 2 == 0:
                    print(f'O número é par {num} é PAR')
                else:
                    print(f'O número {num} é IMPAR')
